Dashboard

The FastAPI dashboard exposes operational endpoints and simple browser views for monitoring positions, trades, signals, health, and current account state.

dashboard endpointsLOCALHOST

The dashboard gives the operator a live readout without needing to parse logs. It should be protected when exposed beyond localhost.

EndpointPurpose
/healthBasic service status for uptime checks and systemd monitoring.
/dashboardMain browser page with bot state, positions, equity, and recent trades.
/api/positionsCurrent holdings, side, quantity, average price, and unrealized P/L.
/api/tradesRecent executions and strategy attribution.
/api/signalsLatest strategy outputs and resolver decisions.
uvicorn monitoring.dashboard:app --host 0.0.0.0 --port 8000

For real-time viewing, the dashboard can poll JSON endpoints or evolve toward WebSockets when live streaming becomes necessary.
