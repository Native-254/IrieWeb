Deployment

Deployment turns the bot into a managed VPS service with Docker packaging, systemd supervision, secret handling, logging, and basic security hardening.

deployment guideALWAYS ON

A VPS setup should favor repeatability and recovery. The bot needs to boot cleanly, restart after failure, protect secrets, and expose only the ports that are truly needed.

AreaRecommendation
VPSOracle Cloud free-tier ARM instances are a practical target for always-on paper trading and monitoring workloads.
DockerPackage Python dependencies, runtime commands, and environment defaults in a container image.
systemdRun the container or Python process as a managed service with restart policy and journal logs.
SecretsStore broker credentials, webhook URLs, and Telegram tokens outside source control in environment files or secret managers.
SecurityUse SSH keys, firewall rules, least-privilege users, update schedules, and private dashboard access.
